# IMPEX xSeries - Quick Start Guide

## 🚀 Get Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Update Contact Information
Open these files and replace placeholder values:

**Phone:** `+910000000000` → Your actual phone number
**Email:** `sales@impex.example` → Your actual email
**WhatsApp:** `910000000000` → Your WhatsApp number (with country code, no +)

Files to update:
- `src/components/site/CTA.tsx`
- `src/components/site/Footer.tsx`
- `src/components/site/StickyMobileCTA.tsx`
- `src/components/site/EnquiryForm.tsx`

### 3. Run Development Server
```bash
npm run dev
```

Visit `http://localhost:5173` to see your website!

---

## 📦 Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder. Upload these to your web host.
